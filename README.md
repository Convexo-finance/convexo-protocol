# Convexo Protocol - Institutional Website

Professional and institutional website for Convexo Protocol, featuring comprehensive DeFi solutions for LATAM enterprises.

## Features

- **Tokenized Corporate Bonds**: Access international funding with AI-powered credit scoring
- **Digital Assets Treasury**: Trade local and international stablecoins
- **Local Stablecoins**: ECOP for Colombia powered by Uniswap V4 and Chainlink
- **OTC Services**: Professional digital asset trading
- **Payment Solutions**: Stablecoin payments for contractors and suppliers

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Font**: Nunito (Google Fonts)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
